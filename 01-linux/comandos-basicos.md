# Comandos básicos de Linux

## Navegación
| Comando | Qué hace |
|---------|----------|
| `pwd` | Muestra la ruta donde estoy |
| `ls` | Lista archivos y carpetas |
| `ls -la` | Lista todo incluyendo ocultos con detalles |
| `cd ~` | Va a home sin importar dónde estoy |
| `cd ..` | Sube un nivel |

## Archivos y carpetas
| Comando | Qué hace |
|---------|----------|
| `mkdir nombre` | Crea una carpeta |
| `touch archivo.md` | Crea un archivo vacío |
| `mv origen destino` | Mueve o renombra |
| `rmdir nombre` | Elimina carpeta vacía |
| `cat archivo` | Muestra el contenido |
| `nano archivo` | Abre editor de texto |

## Permisos del output de ls -la
| Símbolo | Significado |
|---------|-------------|
| `d` | Es una carpeta |
| `r` | Permiso de leer |
| `w` | Permiso de escribir |
| `x` | Permiso de ejecutar |
| `-` | Sin ese permiso |

## Administración
| Comando | Qué hace |
|---------|----------|
| `sudo` | Ejecuta como superusuario |
| `sudo apt update` | Actualiza lista de paquetes |
| `sudo apt install` | Instala un programa |

## Git — flujo diario
| Comando | Qué hace |
|---------|----------|
| `git add .` | Prepara todos los cambios |
| `git commit -m "msg"` | Guarda un punto en la historia |
| `git push` | Sube a GitHub |
